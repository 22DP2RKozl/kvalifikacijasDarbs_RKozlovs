# Pakalpojumu web-site

### Galvenâ lapa
- îss pakalpojuma apraksts
- Poga, lai pârietu pie pakalpojuma izvçles

### Pakalpojumu izvçle
- pakalpojumu saraksts, ar iespçju iegâdâties to online
- sâkumâ visizdevîgâkie pakalpojumi (vispopulârâkie)

### Atsauksmes
- lietotâji varçs atstât savu atsauksmi par servisu un atseviðíi katram pakalpojumam

### "Mans profils"
- lietotâjiem bûs iespçja piereìistrçties web-lapâ
- bûs saglabâta pirkuma vçsture
- varçs pievienot pakalpojumu "vçlos" sadaïâ savâ kontâ, lai vçlâk to âtri atrast.
- varçs pievienot savu pakalpojumu
- atseviðía sadaïa profilâ, kas atbildçs par visiem piedâvâtiem pakalpojumiem

### Èats starp pakalpojuma klientu un sniedzçju
- èats bûs online reþîmâ, nevajadzçs atjaunot web-lapu
- èats atrradîsies blakus profila izvelnei 

### "Par mums"
- apraksts par web lapas mçríi un izveidotâju
- web lapas mçríis ir atvieglot savstârpçju sadarbîbu starp cilvçkiem

### "Sazinâties ar mums"
- tâlrunis un epasts, kur var rakstît, jautâjumu gadîjumos
- tehniskais atbalsts (ar botu vai ar adminu)

### Statistika
- cik lietotâji piereìistrçti
